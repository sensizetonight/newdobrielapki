# Веб-приложение "Добрые Лапки" — Система управления приютом для животных

## Описание проекта

Веб-приложение для управления приютом животных с полным функционалом учета, аналитики и взаимодействия с посетителями. Система позволяет эффективно управлять животными, пожертвованиями, заявками на усыновление и предоставляет администраторам мощный аналитический инструментарий для принятия обоснованных решений.

## Технологический стек

### Backend
- **Python 3.6+** — основной язык разработки
- **Flask 3.1.0** — веб-фреймворк для создания RESTful API и маршрутизации
- **Flask-Login 0.6.3** — система аутентификации и управления сессиями
- **SQLite3** — реляционная база данных
- **Werkzeug 3.1.3** — безопасное хеширование паролей и утилиты

### Frontend
- **HTML5, CSS3, JavaScript** — базовая разметка и стилизация
- **Chart.js 4.4.0** — интерактивные графики и визуализация данных
- **Font Awesome 6.4.0** — векторные иконки
- **Google Fonts (Inter)** — современная типографика
- **Yandex Maps API** — интерактивные карты с геолокацией

### Архитектура
- **MVC (Model-View-Controller)** — паттерн проектирования
- **RESTful API** — эндпоинты для получения аналитических данных
- **Адаптивный дизайн** — поддержка мобильных устройств

## Основной функционал

### 1. Управление животными
- **Каталог животных** с фотографиями и подробной информацией
- **CRUD-операции** для администраторов и волонтеров
- **Система заявок на усыновление** с отслеживанием статусов
- **Фильтрация** по статусу и категориям
- **Загрузка и управление фотографиями** животных

### 2. Финансовая система
- **Прием пожертвований** (на приют или конкретное животное)
- **Учет инвентаря** с категоризацией (корм, медикаменты, расходные материалы)
- **Автоматический расчет** общей стоимости инвентаря
- **Финансовая отчетность** с детализацией по периодам

### 3. Аналитический дашборд
- **Визуализация пожертвований** — временные ряды с фильтрацией (7/30/90 дней)
- **Тренды усыновлений** — анализ динамики заявок
- **Распределение инвентаря** — круговые диаграммы по категориям
- **Прогнозирование затрат** — алгоритм линейной регрессии для предсказания расходов
- **Финансовые сводки** — общая статистика доходов и расходов

### 4. Интерактивная карта
- **Геолокация бездомных животных** с метками на карте
- **Добавление/удаление меток** администраторами и волонтерами
- **Информационные окна** с описанием и фотографиями животных
- **Интеграция с Yandex Maps API**

### 5. Система новостей
- **Публикация новостей** приюта
- **Управление контентом** для администраторов
- **Отображение на главной странице**

### 6. Административная панель
- **Управление заявками** на усыновление (одобрение/отклонение)
- **Управление пожертвованиями** и инвентарем
- **Фильтрация и поиск** по различным критериям
- **Логирование действий** пользователей

## Структура базы данных

Система использует **8 взаимосвязанных таблиц**:

1. **users** — пользователи системы (id, username, password_hash, role)
2. **animals** — животные в приюте (id, name, description, status, age, gender, photo_url)
3. **adoption_requests** — заявки на усыновление (id, animal_id, user_name, phone, email, status)
4. **donations** — пожертвования (id, user_name, amount, donation_type, animal_id, donated_at)
5. **inventory** — инвентарь приюта (id, item_name, category, quantity, unit_price)
6. **news** — новости приюта (id, title, content, date)
7. **stray_animals** — бездомные животные на карте (id, description, lat, lon, photo_url)
8. **logs** — логи действий пользователей (id, action, description, user_id, created_at)

## Реализованные алгоритмы

### Прогнозирование затрат
Алгоритм использует **линейную регрессию** для предсказания будущих расходов на основе исторических данных о пожертвованиях и инвентаре. При недостатке данных система использует средние значения для обеспечения стабильности прогнозов.
