## Требования
- Python 3.6+
- SQLite (встроен в Python)

## Установка
1. Распакуйте проект в папку на сервере.

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

3. Инициализируйте базу данных:
   ```
   python create_db.py
   ```
    
4. Запуск
   ```
   python app.py
   ```

## Новые функции

### 📊 Аналитический дашборд
Полнофункциональный аналитический дашборд с визуализацией данных:
- **Финансовая аналитика**: общие доходы, расходы, баланс
- **Динамика пожертвований**: график пожертвований за период
- **Тренды усыновлений**: статистика по заявкам на усыновление
- **Анализ инвентаря**: распределение по категориям
- **Прогнозирование затрат**: прогноз расходов на 3 месяца вперед
- **Статистика по животным**: распределение по статусам
- **Анализ пожертвований**: разбивка по типам (на приют / на животное)

### 🎨 Современный дизайн
- Обновленный UI/UX с градиентами и современными цветами
- Адаптивный дизайн для всех устройств
- Плавные анимации и переходы
- Улучшенная типографика

### 🔧 Улучшения функциональности
- API endpoints для аналитики (`/api/analytics/*`)
- Интерактивные графики с Chart.js
- Фильтры по периодам (7, 30, 90 дней)
- Детальная финансовая отчетность

## Доступ к аналитике
Аналитический дашборд доступен только администраторам:
- URL: `/admin/analytics`
- Или через меню навигации (кнопка "Аналитика")

## Учетные данные по умолчанию
- **Администратор**: `admin` / `admin123`
- **Волонтер**: `volunteer1` / `volunteer123`
- **Гость**: `guest1` / `guest123`
