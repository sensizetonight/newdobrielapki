# Краткая сводка проекта "Добрые Лапки"

## Быстрая справка для курсовой работы

### Стек технологий

**Backend:**
- Python 3.6+
- Flask 3.1.0 (веб-фреймворк)
- Flask-Login 0.6.3 (аутентификация)
- SQLite3 (база данных)
- Werkzeug 3.1.3 (безопасность)

**Frontend:**
- HTML5, CSS3, JavaScript
- Chart.js 4.4.0 (графики)
- Font Awesome 6.4.0 (иконки)
- Google Fonts (Inter)

**База данных:**
- SQLite (8 таблиц)
- Реляционная модель данных

### Основной функционал

1. **Управление животными**
   - Каталог животных
   - Добавление/редактирование/удаление
   - Подача заявок на усыновление

2. **Финансы**
   - Прием пожертвований
   - Учет инвентаря
   - Финансовая отчетность

3. **Аналитика**
   - Графики пожертвований
   - Тренды усыновлений
   - Анализ инвентаря
   - Прогнозирование затрат

4. **Карта**
   - Отметки бездомных животных
   - Геолокация

5. **Администрирование**
   - Управление заявками
   - Управление инвентарем
   - Аналитический дашборд

### Роли пользователей

- **admin** - полный доступ
- **volunteer** - работа с животными и картой
- **guest** - просмотр и заявки

### Ключевые особенности

✅ Система аутентификации и авторизации
✅ Аналитический дашборд с визуализацией
✅ Прогнозирование затрат (линейная регрессия)
✅ RESTful API для аналитики
✅ Адаптивный дизайн
✅ Безопасность (хеширование паролей, защита от SQL-инъекций)
✅ Логирование действий

### API Endpoints

- `GET /api/analytics/donations-timeline?days=30`
- `GET /api/analytics/adoption-trends?days=30`
- `GET /api/analytics/inventory`

### Маршруты приложения

**Публичные:**
- `/` - главная
- `/animals` - каталог животных
- `/map` - карта
- `/donate` - пожертвования
- `/contact` - контакты
- `/register` - регистрация
- `/login` - вход

**Защищенные:**
- `/admin/dashboard` - админ-панель
- `/admin/analytics` - аналитика
- `/logout` - выход

### База данных

**8 таблиц:**
1. users - пользователи
2. animals - животные
3. adoption_requests - заявки на усыновление
4. donations - пожертвования
5. inventory - инвентарь
6. news - новости
7. stray_animals - бездомные животные
8. logs - логи действий

### Алгоритмы

1. **Прогнозирование затрат**
   - Линейная регрессия
   - Анализ исторических данных
   - Прогноз на 3 месяца

2. **Финансовый анализ**
   - Расчет баланса
   - Статистика по периодам
   - Анализ трендов

### Безопасность

- Хеширование паролей (PBKDF2)
- Защита от SQL-инъекций
- Валидация входных данных
- Контроль доступа по ролям
- Логирование действий

### Установка и запуск

```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Инициализация БД
python create_db.py

# 3. Запуск
python app.py
```
